Web Development Notes

HTML

  • Javascript executed using the <script> element is inserted into the DOM exactly where the <script> element is located in the HTML document/file. 
                        
                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Document</title>
                            <script>Script 1</script> /* -- script 1 will be executed in head element -- */
                        </head>
                        <body>
                            <script>Script 2</script> /* -- script 2 will be executed in body element -- */
                        </body>
                        <footer>
                            <script>Script 3</script> /* -- script 3 will be executed in footer element -- */
                        </footer>
                        </html>

CSS

  • Move text in <pre> element of class "html" with child <code> element to the left 100 pixels using external CSS stylesheet. 
                        
                        pre.html > code {
                            position: relative; left: -100px;
                        }

Javascript

  • Create HTML element form Javascript array 
                        
                        var list = 'alpha,bravo,charlie'.split(',');
                        for (var i in list) {
                            var e = document.createElement('div');
                            e.id = list[i];
                            e.className = "customClass";
                            e.innerHTML = list[i];
                            document.body.appendChild(e);
                        }
