Git
  • Delete local branch 
    
git branch -d <branch name>
  • Delete remote branch 
    
git push origin -d <remote branch name>
  • Rename local branch 
    
git branch -m <new branch name>
  • Compare (diff) two Git branches 
    
git diff <branch 1 commit hash> <branch 2 commit hash>
  • Git merge
    • Merge a branch (branch 2) into another branch (branch 1) 
      
git checkout <branch 1 name>
git merge [--ff, --squash] <branch 2 name>
    • Do a merge for evaluation and without actually commiting it 
      
git merge --no-commit --no-ff <branch name>
      • Examine (diff) results of merge (staged changes) 
        
git diff --cached
      • Abort/backout merge 
        
git merge --abort
  • View Git log
    • View log of branch that is currently checked out 
      
git log
    • View log of "main" branch when not checked out 
      
git log main..
    • View log of a branch NOT checked out 
      
git log main..<branch name>
  • Create a new GitHub repo from someone else's repo
    • Create the new empty repo in GitHub
    • Make a bare clone of someone else's repo in a temp local location 
      
git clone --bare https://someone.elses.repo.git
    • Change directory into the newly cloned bare repo 
      
cd someone.elses.repo.git
    • Perform a mirror push to your new remote GitHub repo 
      
git push --mirror https://my.github.repo.git
    • clean up/remove temp bare repo 
      
cd ..
rm -rf someone.elses.repo.git
